code:
            
                <svg class="myborder" width="200px" height="200px">
                   <line x1="10" y1="10" x2="100" y2="100"></line>
                </svg>
            
        A description of the element's attributes is as follows:
        • x1 and y1: The starting x and y coordinates
        • x2 and y2: The ending x and y coordinates
        • stroke: This gives the line a red color
        • stroke-width: This denotes in pixels the width of the line to be drawn

            The line tag also has the ability to change the style of the end of the line. For
        example, adding the following would change the image so it has round endings:
            stroke-linecap: round;
            As stated earlier, all SVG tags can also be styled with CSS elements. An alternative
        way of producing the same graphic would be to first create a CSS style, as shown in
        the following code:
            line {
                stroke: red;
                stroke-linecap: round;
                stroke-width: 5;
            }
